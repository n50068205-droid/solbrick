# 🏗️ SolBrick — Токенизация недвижимости на Solana

## 🎯 Проблема
В Казахстане купить квартиру или коммерческую недвижимость стоит от $50,000+.
Обычный человек не может инвестировать в недвижимость с маленьким капиталом.
Рынок непрозрачный, посредники берут комиссии, активы неликвидны.

## 👤 Для кого
- Молодые люди 18-35 лет с накоплениями от $25
- Люди которые хотят инвестировать в недвижимость но не могут купить целый объект
- Инвесторы которые хотят прозрачности и ликвидности

## 💡 Решение
SolBrick токенизирует строительные объекты — каждый объект делится на доли (токены).
Любой человек может купить долю от $25 и стать совладельцем реального актива.

## ⚡ Почему Solana
- **Ownership on-chain** — право собственности записано в блокчейн
- **Payments** — покупка долей через смарт-контракт
- **Transparent state** — все транзакции публичны и проверяемы
- **Без Solana проект сломается** — вся логика владения и передачи в смарт-контракте

## 🔧 MVP — Один работающий core flow
1. Пользователь подключает Phantom Wallet
2. Видит список строительных объектов с прогрессом продаж
3. Покупает долю (on-chain транзакция через Anchor смарт-контракт)
4. Становится совладельцем объекта

## 🏗️ Архитектура
```
Пользователь → Phantom Wallet → React Frontend
                                      ↓
                              Anchor Program (Solana)
                                      ↓
                    [create_asset] [buy_shares] [get_holding]
                                      ↓
                              Solana Devnet
```

## 📦 Технологии

- **Program ID**: `2vi5oMpNoo7Qvbjf9sYM3rr4NhZzQnR5ViSsuX1wErG3`

## 🚀 Запуск
```bash
# Смарт-контракт
cd solbrick
anchor build
anchor deploy

# Фронтенд
cd app
npm install
npm start
```
