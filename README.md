# 🏗️ SolBrick

Платформа фракционного владения строительными проектами на Solana.

## Проблема
- Высокий порог входа в недвижимость
- Низкая ликвидность реальных активов
- Отсутствие прозрачности владения

## Решение
SolBrick позволяет инвестировать в строительные проекты от $25, получая токены-доли на Solana.

## Технологии
- **Blockchain**: Solana (Devnet)
- **Smart Contract**: Anchor Framework (Rust)
- **Frontend**: React + TypeScript
- **Wallet**: Phantom Wallet

## Функционал MVP
- Создание токенизированных строительных объектов
- Покупка долей (фракционное владение)
- Подключение Phantom Wallet
- Отображение прогресса продаж и ROI

## Запуск
```bash
cd app
npm install
npm start
```
